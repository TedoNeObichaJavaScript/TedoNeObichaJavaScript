### About me

- 🚀 Founder of [**Nooze**](https://github.com/Nooze-Alarm) — a no-snooze Android alarm app with smart wake, gamification, and smart-home integration.
- 🛠️ Daily driver: **Kotlin · Jetpack Compose · Firebase · Cloud Functions · Gradle**.
- 🌱 Currently exploring **Kotlin Multiplatform** and **Compose Multiplatform** to take Nooze cross-platform.
- 🤖 On the side — building Claude-powered dev tools and custom MCP servers in Node.js to speed up my own workflow.
- 🧠 Orchestrating up to **8 Claude Code agents in parallel** — splitting one task into independent threads (research / impl / tests / review) for concurrent shipping.
- 🌐 Live product: [noozealarm.com](https://noozealarm.com)
